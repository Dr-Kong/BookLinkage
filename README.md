# bookLinkage
***BookLinkage***是由深圳高级中学在校生自主开发、为校内师生提供二手书籍信息的公益微信小程序。平台目前向深高国际部开放，旨在环保利用高年级同学们使用过的书籍，解决低年级同学们无处购买教材的难题。由于腾讯官方禁止个人开发者在小程序中实现付款功能，平台不涉及资金交易，只提供书本上传者的手机或微信号，书本价格及详细信息需同学们私下协商。
